# Docs — AskAide AI

## What This Repo Is

`askaide-docs/` is the **public documentation website**, built with [Docusaurus](https://docusaurus.io/). It is a static site — no application logic, no access to the product database or backend. Content lives in `docs/`; the built output goes to `build/`.

## Commands

```bash
npm install     # Install dependencies
npm start       # Local dev server with live reload
npm run build   # Generate static site into build/
npm run serve   # Serve the built site locally
```

## Layout

| Path | Purpose |
|------|---------|
| `docs/` | Markdown/MDX documentation pages (the actual content) |
| `src/` | Custom React components, pages, and CSS for the Docusaurus theme |
| `static/` | Static assets served as-is (images, favicon, etc.) |
| `docusaurus.config.js` | Site config — title, nav, URL, deploy settings |
| `sidebars.js` | Documentation sidebar structure |
| `build/` | Generated output — **do not edit by hand** (regenerated by `npm run build`) |

## Guardrails

1. **This site is public.** Never add pricing, sales scripts, funding strategy, credentials, internal audits, or anything from `askaide-internal/` here. Public/private separation is deliberate.
2. **Don't hand-edit `build/`** — it is generated. Edit sources in `docs/` / `src/` and rebuild.
3. Keep content in sync with actual product behavior; stale docs mislead users and hurt SEO.
